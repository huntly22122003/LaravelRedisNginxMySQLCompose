## Branches

- **main**: Infrastructure setup (Laravel backend, Nginx, Redis, MySQL).
- **admin-task-security-log**: Initial REST API for user login and register (localhost), built as a foundation before Shopify OAuth integration.
- **shopify-oauth**: Shopify OAuth flow and access token handling.
- **shopify-polaris**: Full working application using React (Vite) and Shopify Polaris.
- **shopify-nextjs**: Full working application using NextJS(Vercel) and Shopify Polaris to public internet (NextJS is front-next project)
- **NOTICE**: PICTURE_RUN shown the program.

> ⚠️ **Important**: Please use the **shopify-polaris** branch to run and evaluate the complete Shopify app.
